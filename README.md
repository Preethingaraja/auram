# Auram by Shree Bheru Gifts 🎁

> **"Unlock Happiness with us.."**

A premium gifting brand website for **Auram by Shree Bheru Gifts**, based in Salem & Bangalore, India.

## Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Tech Stack

- **Vite + React 18** — Build tooling & UI framework
- **Tailwind CSS v3** — Utility-first styling with brand theme extension
- **framer-motion** — Scroll-triggered section animations
- **lucide-react** — Icon library
- **react-router-dom** — Hash-based SPA routing

## Project Structure

```
src/
  components/
    Navbar.jsx        Sticky frosted-glass nav + mobile drawer
    Hero.jsx          Full-viewport animated hero section
    Products.jsx      6-card product categories grid
    WhyUs.jsx         4-column features + CTA banner
    Branches.jsx      Salem & Bangalore location cards
    Contact.jsx       Contact form + team info + Google Maps
    Footer.jsx        Dark footer with links and copyright
  App.jsx             Root app, dark mode, router
  main.jsx            React DOM entry point
  index.css           Tailwind directives + custom CSS
tailwind.config.js    Brand color theme extension
```
